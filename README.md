# Quiz_App

The Quiz App is an engaging and educational web application designed to test and enhance your knowledge on a wide range of topics. It offers an interactive and fun way to challenge yourself, learn new facts, and compete with others in a quiz format. Whether you're a student preparing for exams, a trivia enthusiast, or just looking to expand your knowledge, this app has something for everyone.

Key Features:

Multiple Choice: Answer questions by selecting from multiple choices, making it accessible to users of all ages and backgrounds.

Score Tracking: Keep track of your scores and compare them with friends or other users on leaderboards.

Educational: Learn interesting facts and gain knowledge on various topics while having fun.

User-Friendly: The intuitive user interface ensures a seamless and enjoyable quiz-taking experience.

Only five questions are added in app.js file you can add more.
